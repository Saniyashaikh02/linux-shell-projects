# Linux Shell Projects

This repository contains multiple Linux shell scripting mini projects.

## 📁 Project 1: Health Monitor

### 🔹 Description

This script monitors system disk usage and logs partitions that exceed 80% usage.

### 🔹 Commands Used

* df -h
* awk
* sed
* bash scripting

### 🔹 How to Run

```bash
cd project1-health-monitor
chmod +x health_monitor.sh
./health_monitor.sh
```

### 🔹 Output

Logs are stored in:

```
system_health.log
```
